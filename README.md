# _Amppper_

A GUI application with the following featues:
  - **Audio Manipulation** (dormant)
    - [x] Trim
    - [x] Join
    - [x] Change format
    - [ ] Change bitrate (proposed)
    - [ ] Audio to text (proposed)
  - **Image Manipulation** (proposed)
    - [ ] Crop
    - [ ] Add filters
    - [ ] Resize
    - [ ] Change format
  - **Video Manipulation** (proposed)
    - [ ] Trim
    - [x] Extract Audio
    - [x] Mute
    - [x] Join (improvement required)
  - **PDF tools** (proposed)
    - [x] Split into pages
    - [x] Convert to text
    - [ ] Read
    - [ ] Text to PDF
    - [x] Join PDF files
